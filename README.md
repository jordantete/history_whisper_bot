Core classes, functions, and methods:

1. HistoricalFigure - a class representing a historical figure with a name and description.
2. Database - a class representing the database containing the historical figures.
3. Bot - a class representing the telegram bot and its functionality.
4. send_message - a function to send a message to all users at once.
5. pick_random_figure - a function to randomly pick a historical figure from the database.
